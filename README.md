# To-Do-List-Manager-
To-Do List Manager is a project that developed as final project of BIM209. It aims to facilitate the organization of complex tasks and simplify the tracking of completed and pending assignments for individuals with busy lives. It is designed to aid in time management and ultimately help users save
time. Since the system operates in a digital environment, it allows individuals to access their lists
anytime and anywhere, ensuring that their tasks and commitments are not overlooked or delayed. It is a
software system that enables users view, create and delete tasks from their lists. Tasks can be marked as
completed and users can view pending or completed lists. It is developed using Java language and
Object Oriented Design Principles. Project is designed to include decided use cases. This system will
be beneficial for students who has engaged academic lives, professionals who wants to organize their
daily tasks and business owners who needs tracking systems.The primary benefit is centralizing task
management to enable users to focus, prioritize, and ensure that no important commitment is forgotten.
Users will be more motivated by seeing their progress instantly and will be able to use their time more
effectively. This translates into a concrete increase in overall productivity.
